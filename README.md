# API RESTful Full Stack com Laravel e React

Este projeto demonstra o desenvolvimento de uma API RESTful utilizando Laravel (PHP) e sua integração com um frontend desenvolvido em React (Vite).

## Tecnologias Utilizadas

- Laravel (PHP)
- React JS
- Vite
- MySQL
- Insomnia
- Git e GitHub

## Funcionalidades

- Desenvolvimento de API RESTful
- Operações CRUD (Criar, Ler, Atualizar e Excluir)
- Consumo de API no Frontend
- Requisições HTTP
- Gerenciamento de Rotas
- Persistência de Dados

## Testes da API

Todos os endpoints da API são testados e validados utilizando o Insomnia, permitindo realizar requisições HTTP, validar respostas e auxiliar no processo de depuração durante o desenvolvimento.

## Objetivo do Projeto

O objetivo deste projeto é demonstrar a construção de uma aplicação Full Stack moderna, utilizando Laravel para o desenvolvimento da API e React para a interface do usuário, aplicando boas práticas de desenvolvimento, integração entre frontend e backend e controle de versão com Git e GitHub.
